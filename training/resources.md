# External DFIR Training Resources

Paid and free. Verify links before relying on them — the web moves.

## Reference sites (bookmark these)
- **Aboutdfir** — artifact reference, case studies, cheat sheets
- **The DFIR Report** — real intrusion walkthroughs with IOCs and timelines
- **SANS DFIR cheat sheets** — free PDFs: Windows artifacts, memory, network, cloud
- **13Cubed YouTube** — deep, practical Windows forensics
- **MalwareAnalysisForHedgehogs** — RE-focused YouTube
- **Eric Zimmerman's tools page** — the standard Windows artifact parsers (KAPE, Registry Explorer, EZ tools)

## Free / low-cost labs
- **TryHackMe — DFIR / SOC Level 1 & 2 paths**
- **Blue Team Labs Online (BTLO)** — DFIR-heavy challenges
- **CyberDefenders** — enterprise-style blue team CTFs
- **MalwareTech labs** — malware RE practice
- **DFIR Madness** — case walkthroughs + evidence sets
- **Magnet Weekly CTF archive** — past challenges with writeups
- **CTF Time (filter by forensics)** — find upcoming forensic CTFs

## Paid courses (highest signal first)
- **SANS FOR500** — Windows Forensic Analysis
- **SANS FOR508** — Advanced IR, Threat Hunting, Digital Forensics
- **SANS FOR572** — Network forensics
- **SANS FOR610** — Reverse-engineering malware
- **SANS FOR509** — Cloud forensics
- **Zero2Automated** — malware analysis / RE
- **Offensive Security EXP-312 / PEN-300** — adjacent but useful for attacker mindset

## Reading list
- *The Art of Memory Forensics* — Ligh, Case, Levy, Walters
- *Practical Malware Analysis* — Sikorski, Honig
- *Incident Response & Computer Forensics* — Luttgens, Pepe, Mandia
- *File System Forensic Analysis* — Carrier
- *Windows Internals, 7th Ed.* — Russinovich, Solomon (for memory forensics)
